AoS Playmat Builder

AoS has a welcome page which gives the user the option to log in or to get started right away. Logging in gives access to saving up to 3 projects. The main functionality is in the "builder page" and "playmat page"

Builder Page

The user can select single option categories and add/remove items to multiple category options lists. These are dynamically populated based on choices.

Playmat Page

An appropriately styled list of rules based on user selections. Overall style based on faction choice. Rules listed in game phase order to assist players. Print option simplifies display for easier printing. Option to save playmat if logged in.

User Page

If logged in, user has access to up to three saved playmats which they can name or delete.

Playmat Errors

Error handling for cases where users might try to save over a project or access an empty save, and alert dialogue for deletion confirmation etc.

Nartwhal

Nartwhal is based on a style guide by Irina Patrikeeva Hallinan, a design student. It uses the Cooper Hewitt public exhibit API as a model for exhibition data.

Welcome Page

The Welcome Page provides login functionality.

Explore Page

Wireframe

Actual

The explore page allows users to type in search queries. The results are displayed in a carousel. Each item in the carousel consists of a tile with title, image, and details info. It can be scrolled, and updates based on search criteria.

Preferences Page

Wireframe

Actual

The preferences page allows users to select tags that will filter exhibition results. It has sort and display selected/all preferences functionality.

Settings Page

Wireframe

Actual

The settings page allows users to change their profile info, toggle push notifications, and set the frequency of alerts.

History Page

The history page allows users to view the exhibitions they previously visited. It acts like an exhibition journal.
